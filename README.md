This repository contains instructions and skills files necessary to quickly build SuiteScript custom solutions with the help of GitHub Copilot.

## Installation

1. Clone the contents of this repository to your local machine.
2. Open a new GitHub chat session.

### Plan Mode

Before asking Copilot to write scripts, it is highly recommended to use Plan mode first.

To go into Copilot plan mode, enter the keyword "/plan" first followed by your prompt, as shown in the following example:

```
/plan Create a user event for...
```

This will cause Copilot to design a solution first without writing any code. Copilot may also ask you questions if the requirements are a bit ambiguous or if there are any design considerations to incorporate.

### Implement Mode

At the end of its run, Copilot will show you a **Start Implementation** button. When you are satisfied with the plan, you may click this button to let Copilot execute the plan it wrote.
